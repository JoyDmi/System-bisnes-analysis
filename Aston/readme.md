Задание 1. Описание бизнес-процесса

Общее описание: необходимо смоделировать бизнес-процесс с использованием нотации BPMN, который отображает процесс формирования заявки поставщику с последующим сохранением и изменением статусов заявки в БД SQL Server

Описание бизнес процесса:

Сотрудник департамента закупок оформляет заявку на закупку товара у поставщика в приложении “Оформитель заявок”

Заявка сохраняется в БД SQL Server

Поставщик получает заявку в приложении “Приемщик заявок” и принимает одно из двух решений:

Согласовать исполнение заявки на поставку товара

Отклонить выполнение заявки на поставку товара

Если заявка была согласована в течении 3-х рабочих дней, то в БД SQL Server изменяется статус заявки на “Согласована”

Если поставщик отклонил заявку или она не была согласована в течении 3-х рабочих дней, то в БД SQL Server статус заявки изменяется на “Отклонена”

Задание 2. API

Задача:

Требования:
Создать API для управления товарами:

Получить список всех товаров.
Получить информацию о конкретном товаре по его ID.
Добавить новый товар.
Обновить информацию о существующем товаре.
Удалить товар.
